# Кликер-игра на Express.js

## Описание
Простое кликер-приложение на Express.js, позволяющее пользователю кликать на кнопку для получения очков и покупать улучшения для увеличения эффективности.

## Функциональность
- Клик на кнопку для получения очков
- Покупка улучшений за очки
- Автоматические кликеры (генерируют очки автоматически)
- Статистика игры
- Возможность сброса игры
