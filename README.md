# Кликер-игра на Express.js

## Описание
Простое кликер-приложение на Express.js, позволяющее пользователю кликать на кнопку для получения очков.

## Функциональность
- Клик на кнопку для получения очков
- Автоматические кликеры (генерируют очки автоматически)
- Статистика игры
